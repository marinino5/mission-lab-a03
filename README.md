# Mission Lab A03:2025

Demo web académico y seguro para explicar OWASP Top 10:2025 A03: Fallos en la cadena de suministro de software.

## Archivos

- `index.html`
- `styles.css`
- `script.js`

## Cómo usarlo

1. Abre `index.html` en el navegador.
2. Haz clic en **Simular dependencia comprometida**.
3. Luego haz clic en **Ejecutar controles de seguridad**.
4. Usa los botones de decisión para hacer participar al público.

## Publicar en GitHub Pages

1. Crea un repositorio.
2. Sube estos tres archivos en la raíz del repositorio.
3. Ve a **Settings > Pages**.
4. En **Source**, selecciona `Deploy from a branch`.
5. Elige `main` y `/root`.
6. Guarda y espera el enlace público.

Este demo no ejecuta ataques reales ni recopila información.
